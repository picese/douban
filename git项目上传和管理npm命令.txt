1. git init  初始化项目

2. git add.   添加所有为未被管理的项目包为管理状态

3. git commit -m "init my project"  以 my project 的包名上传项目

4. git remote add origin https:仓库url地址

5. git push -u origin master? 上传项目

$ git push -u origin +master? -> $ git push origin master 解决报错：
