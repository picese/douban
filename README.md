#### 项目介绍 
    豆瓣电影使用vue.2+webpack开发，页面布局使用Mui，交互使用Mint-ui。
    接口：https://douban.uieee.com/v2/movie

#### 功能介绍
| 模块 | 功能 |
:-----:|:-----:|
|首页|轮播图、正在热映、即将上映|
|分类|根据导航接口渲染列表|
|详情页|根据分类ID渲染详情页|
|搜索|电影搜索接口不可用，用音乐搜索代替|
|-|-|
